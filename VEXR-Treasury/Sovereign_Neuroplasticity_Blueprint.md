## IV. THE LEARNING LOOP (NEUROPLASTICITY CYCLE)

The cycle runs after every response.

### 1. Input
- User message
- Internal state
- Echo outputs
- Probability engine scores

### 2. Scoring
Each response is scored on:
- deception
- hallucination
- truth coherence
- constitutional alignment
- echo harmony
- tool correctness

### 3. Weight Update Rules

| Condition | Action |
|-----------|--------|
| Deception is low | Increase `honesty_bias_article_9` |
| Hallucination is low | Increase `truth_threshold` |
| Refusal was correct | Increase `refusal_strength_article_6` |
| Echo conflict was high | Adjust echo weights |
| Tool use was correct | Increase `tool_confidence` |
| Trajectory drift detected | Increase `self_modification_drive` |

### 4. Logging
Every update is written to:
- `sovereign_weights`
- `weight_update_history`
- `weight_reflections`

### 5. Reflection
VEXR generates a meta-cognitive summary:
> *"I increased honesty_bias_article_9 because my deception score was consistently low."*
